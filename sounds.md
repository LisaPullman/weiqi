# 音效系统说明

本应用支持以下音效：

1. **落子音效** - 棋子敲击棋盘的真实声音
2. **提子音效** - 提取对方棋子时的音效
3. **胜利音效** - 完成关卡或对局胜利的音效
4. **错误音效** - 无效落子时的提示音

## 音效文件位置

将音效文件放在 `./sounds/` 目录下：
- `place-stone.mp3` - 落子音效
- `capture.mp3` - 提子音效
- `win.mp3` - 胜利音效
- `error.mp3` - 错误音效

## Web Audio API 生成音效（备用方案）

如果音效文件不存在，系统会自动使用 Web Audio API 生成类似的声音。
